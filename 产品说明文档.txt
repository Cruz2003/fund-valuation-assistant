基金实时估值助手（Fund Tracker）开发指南
一、项目背景

很多基金投资者希望在盘中实时了解基金的涨跌情况，而不是等到晚上基金公司公布净值后才能知道结果。

例如：

富国互联网科技
易方达蓝筹精选
中欧医疗健康
宝盈纳斯达克100

这些基金每天只公布一次净值。

实际上基金持有的股票在交易时间内是实时波动的，因此可以根据基金持仓和股票实时行情，对基金净值进行实时估算。

本项目旨在开发一个桌面版基金实时估值助手，实现：

自定义添加基金
自动获取基金持仓
自动获取实时股票行情
实时估算基金涨跌
分析每只股票对基金涨跌的贡献
二、项目目标

开发一个本地运行的软件。

用户输入基金代码：

例如：

005827
110022
163402

获取基金信息
↓

↓

↓
展示结果

三、技术选型
开发语言

Python 3.12

GUI框架

PySide6

原因：

免费
功能强大
支持桌面软件
支持打包 EXE
数据分析

Pandas

NumPy

数据库

SQLite

无需安装数据库服务器。

数据源

AkShare、美股和港股数据

用于获取：

基金列表
基金信息
基金持仓
历史净值
四、开发环境
创建虚拟环境

必须使用独立虚拟环境开发。

conda create -n fund_tracker python=3.12 -y

激活环境：

conda activate fund_tracker

安装依赖：

pip install pyside6
pip install pandas
pip install numpy
pip install akshare
pip install requests
pip install matplotlib
pip install openpyxl

导出依赖：

pip freeze > requirements.txt
